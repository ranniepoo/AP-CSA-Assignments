# FirstNames

## Description
This Java program demonstrates several common methods of an `ArrayList` using a list of first names.  
It includes methods for adding, removing, changing, displaying, and checking the size of the list.  
The program runs through a series of test actions in the `main` method to show how each method works.

---

## Concepts Used
- ArrayList data structure  
- Method creation and overloading  
- Index-based operations  
- Conditionals (`if` / `else`)  
- Global variables and access modifiers  

---

## Methods Overview
| Method | Parameters | Return Type | Description |
|---------|-------------|--------------|--------------|
| `setName(String name)` | 1 String | `void` | Adds a new name to the list. |
| `removeName(int index)` | 1 int | `void` | Removes a name from a given index. |
| `changeName(int index, String name)` | 1 int, 1 String | `void` | Changes a name at a given index. |
| `getName()` | none | `void` | Displays all names in the list. |
| `getName(int index)` | 1 int | `String` | Displays one name from the list. |
| `getArraySize()` | none | `int` | Returns the total number of names in the list. |
